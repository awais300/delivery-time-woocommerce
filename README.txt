Delivery Time for WooCommerce will let you easily add delivery time for your products.

== Description ==

Delivery Time for WooCommerce will let you easily add delivery time for your products globally and also on per product level. Optionally you can show these option either on single product page or product archive page.

== Settings == 

* Global settings: WooCommerce -> Settings -> Shipping -> Delivery Time
* Per product settings: All products -> Edit product -> product Data -> Shipping

== Requirements ==

* PHP 7.4
* MySQL 5.7

== Installation ==

1. Upload and unzip `delivery-time-woocommerce.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
